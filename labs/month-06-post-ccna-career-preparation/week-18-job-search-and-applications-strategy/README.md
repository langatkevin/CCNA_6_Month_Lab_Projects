# Week 18 - Job Search and Applications Strategy

> Adapted from the "6-Month CCNA 200-301 Study Plan" reference.

## Objectives
- Identify job leads that fit the desired role and location.
- Customize cover letters or intro emails for each lead.
- Track application status in a simple Kanban or spreadsheet.
- Stay active in networking communities or forums.

## Hands-On Labs & Mini Projects
- Pipeline Task 18.1 - Build an application tracker with stages.
- Pipeline Task 18.2 - Draft a reusable cover-letter template.
- Community Task - Post or share one helpful networking insight.

## Milestones & Checkpoints
- Minimum of five tailored applications sent this week.
- Tracker updated daily with responses or follow-ups.
- Community engagement logged (forum, LinkedIn, meetups).
