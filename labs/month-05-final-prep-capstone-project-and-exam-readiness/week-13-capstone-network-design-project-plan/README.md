# Week 13 - Capstone Network Design Project - Plan

> Adapted from the "6-Month CCNA 200-301 Study Plan" reference.

## Objectives
- Select a realistic network scenario for the capstone project.
- Gather business, capacity, and security requirements.
- Design high-level and low-level diagrams.
- Create a bill of materials plus an addressing and VLAN plan.

## Hands-On Labs & Mini Projects
- Design Task 13.1 - Draft physical and logical diagrams.
- Design Task 13.2 - Build an addressing spreadsheet and VLAN list.
- Risk review - List constraints, open questions, and test plans.

## Milestones & Checkpoints
- Capstone charter approved with scope, success criteria, and timeline.
- Design packet (diagrams, IP plan, requirements) stored under the week folder.
- Test checklist created for the implementation week.
